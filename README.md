# REST API

## 🤖 Workflow connection:

1. Models
2. UseCases
3. Router
4. Server

## 📑 Technologies:

- Javascript
- Node.JS
- Express.JS
- MongoDB
- PNPM

## 🛠️ Installation

1. Clone the repository
2. Install dependencies using: `pnpm install`
3. Create an .env file

## 🏃 Run Scripts

Production mode `pnpm start`
Development mode `pnpm dev`

Send requests to [http://localhost:8080](http://localhost:8080)
